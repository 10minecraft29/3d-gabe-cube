## Description
This is a 3D Gabe Cube that you can move around in any direction that uses studio.hdr for lighting.
The Gabe Cube is fairly resource intensive for a web page.
## Minimum Specs
  - A Modern Web Browser
  - A GPU that supports shader processing and OpenGL (Roughly DirectX 9 level)
  - 512 MB VRAM
  - 4 GB of system memory
## Recomended Specs
  - A modern web browser
  - A modern dual core CPU or greater
  - A GPU that supports OpenGL 2.0
  - 1 GB VRAM or greater
  - 8 GB of system memory or greater
